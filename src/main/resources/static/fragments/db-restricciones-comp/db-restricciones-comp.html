El Código del Producto No se puede repetir. Está permitido solo letras minúsculas, mayúsculas, números , espacios y guiones medios

Ejemplo : UT7-4545

La URL de la Imagen No se puede repetir. Está permitido solo formato URL

Ejemplo : https://getbootstrap.com/docs/5.1/utilities/sizing/

El Nro. de Pieza del Producto Se puede repetir. Está permitido solo letras minúsculas, mayúsculas, números , espacios y guiones medios

Ejemplo : 888A7SASD
