Servicio Componentes

Podemos Visualizar el Listado de Componentes con formato Paginado y obtener información desde la Base de Datos. El Paginado nos permite no saturar la base de datos con muchas peticiones

La Operación de Agregar un Componente se realiza a través del Formulario correspondiente. El mismo posee un Servicio de Ayuda que nos permite identificar que tipo de valor podemos ingresar y cuáles no. De todas formas el Sistema posee filtros para su inserción

Se puede Seleccionar el Componente deseado desde el Listado de Componentes y Editarlo según el valor deseado.Al igual que el Formulario de inserción, existe comprobación y restricción de valores para su ingreso.

El Nro. de Pieza del Producto SI se puede repetir y se debe indicar el valor en el formulario. Está permitido solo letras minúsculas, mayúsculas, números , espacios y guiones medios

Ejemplo : Microcontroladores PICS

La Descripción del Producto SI se puede repetir y se debe indicar el valor en el formulario. Está permitido solo letras minúsculas, mayúsculas, números , espacios y guiones medios

Ejemplo : Placa Wemos D1 Mini

El Fabricante del Producto SI se puede repetir y se debe indicar el valor en el formulario. Está permitido solo letras minúsculas, mayúsculas, números , espacios y guiones medios

Ejemplo : Espressif Systems
